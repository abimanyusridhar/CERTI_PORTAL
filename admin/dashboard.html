    Expected CSV Format
    Your CSV should have these exact column headers:
    IMO Number
    VESSEL_NAME
    Chif Name
    ISSUES_DATES
    Also recognised:
    imo_number, vessel imo
    vessel name, ship name
    chief name, chief engineer
    issue date, compliance date
    recipient_email, email
    VESSEL_NAME — full name as-is:
    MV - PALADIN TRADER
    MT - BW CHINOOK
    Cert ID auto-generated:
    CST-{IMO}-{MM}-{YY}

    Date formats accepted:
    30-Jan-26 · 30/01/2026 · 2026-01-30

    3. Drag & drop or upload the certificate PNG in the image section
    4. Save — the image will show on the public portal
